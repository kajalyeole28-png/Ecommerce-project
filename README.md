End-to-End Data Analysis using Python & SQL
This project demonstrates a complete analytics workflow on an e-commerce dataset — from data extraction and SQL database creation to advanced queries and business insights. It’s designed to showcase real-world analytical skills valuable for data analyst roles.

Project Overview
Objective:
Analyze the sales performance of an e-commerce business by extracting insights from order and customer data using SQL and Python. The goal is to reveal trends, customer behavior, and key business metrics that support decision-making.

Tools used:
Python (Pandas, NumPy)
SQL
Jupyter Notebook
